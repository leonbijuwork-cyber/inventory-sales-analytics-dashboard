# 📊 Inventory Sales Analytics Dashboard (SQL + Power BI)

##  Project Overview
Built an end-to-end data analytics solution using MySQL and Power BI to analyze inventory and sales performance. Designed a relational database, performed data extraction and transformation using SQL, and developed an interactive dashboard to deliver actionable business insights.

---

##  Key Features
- Designed normalized relational database with multiple tables and relationships
- Performed data extraction, filtering, and aggregation using SQL queries
- Built interactive Power BI dashboard with dynamic filters and KPIs
- Enabled time-based analysis (Month, Quarter, Week)
- Implemented sales representative-level performance tracking

---

##  KPIs Tracked
- Total Sales  
- Total Orders  
- Average Order Value  
- Total Quantity Sold  
- Monthly Sales Performance  

---

##  Key Insights
- Identified top-performing product categories driving revenue  
- Analyzed sales trends over time to detect performance patterns  
- Evaluated sales representative performance and contribution  
- Observed customer purchasing behavior across regions.

---

## 🔹 Business Impact
- Enables stakeholders to monitor sales performance in real-time  
- Supports data-driven decision making  
- Improves visibility into sales trends and operations  

---

##  Tools & Technologies
- **SQL (MySQL)** – Database design & querying
